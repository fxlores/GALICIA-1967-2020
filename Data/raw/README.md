# Raw data

This folder contains the original data file used in the replication:

- `datBDMORES.mat`

This file stores the subset of original series required for the empirical application. It is the input read by `code/01_data_preparation/preparation.m`.

The original series are taken from BDMORES base 2015 (May 2023). Definitions of the variables and the details of the data construction are provided in the **Data** section of the paper.

The file in this folder should not be modified manually. All transformations used in the quantitative analysis are generated by the preparation script and saved in `data/processed/`.
