# Processed data

This folder contains the processed files generated by  
`code/01_data_preparation/preparation.m`. 

Starting from the original data file `data/raw/datBDMORES.mat`, the script constructs the model-ready series used in the quantitative analysis. These transformations include deflation, per-capita conversion, detrending, recovery of the capital stock path, computation of wedges, and construction of the transition paths required by Dynare. The script also assigns values to the model parameters used in the paper. 

In particular, `preparation.m` generates two processed files:

- **`PathsGalicia`**: stores the time paths of the model variables and wedges, including `A`, `pi_h`, `pi_f`, `pi_x`, `pi_n`, `pi_g`, `k`, `y`, `x`, `l`, `c`, and `sl`. 
- **`EEGalicia`**: stores the steady-state and initialization objects `ye`, `xe`, `ce`, `le`, `ke`, `A_bar`, `pi_h_bar`, `pi_f_bar`, `pi_x_bar`, `pi_g_bar`, `pi_n_bar`, `sle`, and `K0`. 

These files are not raw data. They are derived objects produced by the replication pipeline and should be fully reproducible by running `preparation.m`. 

Definitions of the original variables and the details of the data construction are provided in the **Data** section of the paper.
