# Processed data

This folder contains the processed files generated by  
`code/01_data_preparation/preparation.m`.

The script reads the original data from `data/raw/datBDMORES.mat` and transforms them into model-ready inputs through deflation, per-capita conversion, detrending, wedge construction, and extension of transition paths for Dynare. 

The script generates two output files:

- **`PathsGalicia`**: stores the series `A`, `pi_h`, `pi_f`, `pi_x`, `pi_n`, `pi_g`, `k`, `y`, `x`, `l`, `c`, and `sl`.
- **`EEGalicia`**: stores the steady-state and initialization objects `ye`, `xe`, `ce`, `le`, `ke`, `A_bar`, `pi_h_bar`, `pi_f_bar`, `pi_x_bar`, `pi_g_bar`, `pi_n_bar`, `sle`, and `K0`. 

These files are derived from the original data and should be reproducible by running the preparation script. The definitions of the source variables and the economic interpretation of the constructed series are given in the **Data** section of the paper.
