# GALICIA-1967-2020
Code and materials for replicating "DRIVERS OF REGIONAL ECONOMIC DEVELOPMENT: TECHNOLOGICAL CHANGE VERSUS MARKET DISTORTIONS IN GALICIA 1967-2020"
# Original data
The original data is taken from the BDMORES base 2015 (Mayo 2023), which is available on the
[Fundación Rafael del Pino’s website.](https://frdelpino.es/investigacion/economia-espanola/economia-regional-espanola/)
# Data processed
# Calibration Scripts
The code that assigns values to the model parameters (such as γ=0.0228 or ρ=−0.388).
# Wedge Calculation Scripts
The code that solves the system of equilibrium equations (1–9) to obtain the wedge values.
# Simulation Scripts
The files that run the ‘non-wedge’ experiments (non-wedge components) and calculate the statistics.
