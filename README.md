# GALICIA-1967-2020
Code and materials for replicating "DRIVERS OF REGIONAL ECONOMIC DEVELOPMENT: TECHNOLOGICAL CHANGE VERSUS MARKET DISTORTIONS IN GALICIA 1967-2020"
# ORIGINAL DATA
The original data is taken from the BDMORES base 2015 (Mayo 2023), which is available on the Fundación Rafael del Pino’s website.
[BDMORES base 2015 (Mayo 2023)](https://frdelpino.es/investigacion/economia-espanola/economia-regional-espanola/)
